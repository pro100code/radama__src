2. Photoshop
   Как доставать картинки
   1-й метод
   выделяем фон => меняем название с форматом изображения => ПКМ => преобразовать в смарт-обьект => файл => ренерировать => набор изображения

   2-й метод 
   ЛКМ на картинку => выделяем все слои => ПКМ => дубликат слоев => новый => файл => выбрать картинку => экспортировать => сохранить для web
