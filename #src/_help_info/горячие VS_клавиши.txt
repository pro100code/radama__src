Общие для VS Code
CTRL+SHIFT+P - найти и выполнить все команды (палитра команд)
CTRL+SHIFT+M - окно с терминалом и консолью
CTRL+` - терминал

 CTRL + F => в окне пишем нужный участок кода, который хотим заменить => галочка вниз => пишем на что заменить => ENTER (первый в коде) или CTRL + ALT + ENTER (заменить все такие участки)

ПАНЕЛЬ ДЕЙСТВИЙ - 4 значка слева
ОБЛАСТЬ РЕДАКТОРА - место для кода
F11 - скрыть верхнюю панель
ALT+Z - перенос строки (включить / отключить)

Мои настройка сочетания клавиш
Управление > сочетание клавиш > в строке поиска вводим плагин > устанавливаем сочетание клавиш (ПКМ - удалить сочетание, если нужно)

1. eCSStractor for VSCode - перенос классов с HTML в CSS 
    Ctrl + Alt + 1 - перенос по BEM с коментириями названия классов
  
2. Alt + W - обвернуть участок кода в тэг 

3. BEM Helper - быстро создает классы по методике BEM
    ALT+B E
    ALT+B M
    ALT+B S

4. Vscode Google Translate - выделеный руский текст переведется на английский
    ALT + SHIFT + T 

5. CSS Navigation - связывает html и css файлы и помогает верстать классы и всплывающие окна помощи
    F12 - кликаешь на название класса в html, жмешь F12 и переходишь на этот класс в css
    ALT+F12 - всплывающее окно, в котором сразу можно менять свойства и значения (если в файле css несколько классов с одним названием)
