Адаптивная HTML-верстка макета из Figma с подробным объяснением. Советы от профи. HTML SCSS GULP. 
https://icomoon.io/app/#/select => Selection(0) - внизу по центру => бургер (слева вверху) => New Empty Set =>
=> бургер (появился справа) => Import to Set и загружаем иконки => выделяем их(желтая стрелка вверху) =>
=> если есть цвета, то удаляем их (возле правого бургера слева) => 
=> Generate Font (справа внизу) и потом эмем тут же на шестеренку =>
=> настройки: 
FONTS NAME - icons
Class Prefix - _icon-
 и CSS СЕЛЕКТОР - второе (Use attribute selectors) =>
=> Названия картинок - это классы для html где автоматически добавится низ-подчеркивание, например _vk (при переустановки проверить настройки на шестеренке) => download в папку fonts =>
=> в папке оставляем только icons.ttf, а с файла style.css копируем названия последних классов со свойствами в icons.scss и удаляем style.css =>
=> теперь если ссылке дать тот класс, то появится картинка
