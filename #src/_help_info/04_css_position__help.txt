========================> Позиционирование <======================
>----------------------------------------------------------------<
position: static; - по умолчанию

position: relative;- делает блок основным для позиционирования дочерних элементов
доп. свойства и селекторы (можно дополнительно указывать для этого блока z-index, left:0, right:0, top:0, bottom:0 либо отрицательное значение)
top:0; или bottom:0;
left:0; или right:0;  
z-index: ... ;  


position: absolute; - вырывает контэнт с общего потока и позиционирует относительно body если у родителя нет position: relative
left:0, right:0, top:0, bottom:0 либо отрицательное значение) - обязательные параметры

position: fixed; - вырывает блок с потока и фиксируется относительно body 

position: sticky; - обьект при скроле прилипнет к верхней части браузера, такой же блок перекроет этот элемент
