7. Проверить установку gulp
На сайте GULP ( https://gulpjs.com/ ) во вкладке "GET STARTED" опускаемся до раздела  "Create a gulp", копируем тот js-код и вставляем в gulpfile.js
Пример кода с этого кода
function defaultTask(cb) {
  // place code for your default task here
  cb();
}

exports.default = defaultTask

Проверяем установку (терминале папки пишем gulp)

8. Плагины 
npm i название плагина --save-dev (чтобы сохранялись версии пакетов)
общие файлы
gulp-plumber                   - чтобы gulp не выключался при ошаике в коде
                                 вставить в начале всех функций перед всеми пайпами — .pipe(plumber())

browser-sync                   -  для отслеживания изменения кода в браузере

gulp-file-include              - для сборки разных файлов в один
                               Называем файл через подчеркивание  _header.html 
                               Так подключаем этот файл           @@include('_header.html')

del                            - для удаления ненужных файлов и папок
 
gulp-sass                      - для отслеживания sass-файлов

gulp-autoprefixer              - добавляет вендорные префиксы 

gulp-group-css-media-queries   - соберает по всему css файлу медиа запросы и групеоует их

gulp-clean-css                 - чистит и сжимает css-файл на выходе

gulp-rename                    - для создания 2-х файлов css (сжатий и обычный)

gulp-uglify-es                 - для сжатия js-файлов 

картинки
gulp-imagemin                  - сжимает картинки без потери качества

gulp-webp                      - делает картинки в формате webp

gulp-webp-html                 - подключает webp картинки

gulp-webpcss                   - css для webp, только сначала нужно сделать проверку в js на поддержку webp

gulp-svg-sprite                - делает спрайты (чтобы вызвать вызываем gulp, а потов во втором терминале svgSprite)

шрифты
gulp-ttf2woff
gulp-ttf2woff2
gulp-fonter                   - для файлов .otf в .ttf 
                                для запуска функции пишем otf2ttf, а когда запустим gulp шрифты перевелутся в нужные
Подключение происходит через функция cb и fontsStyle (но нужно создать fonts.scss и подключить, и создать миксин, который будет подключать файл шрифтов)
В fonts.scss подключились шрифты, но нужно изменить третье значение жирности шрифтов
Можно менять все значения только кроме второго, потому что это имя файла и через это значение идет подключение файла
